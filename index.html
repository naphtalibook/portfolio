Dobble - card game

My first Javascript project.

Dobble is a well-known card game. I programmed it for the fun of practising Javascript.
In the game you see two cards, side by side, each of which has 8 different images. One image appears on both cards. Your task is to identify and click on the one image that appears on both cards. The game is against the clock, so go as fast as you can. Good luck!
